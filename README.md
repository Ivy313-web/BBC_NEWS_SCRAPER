# BBC_NEWS_SCRAPER

A Selenium-based Python tool for automated news data extraction and storage.
Developed as a STEM exploration project in Year 11.

Before the start:

**Download**: pip install selenium openpyxl
- **ChromeDriver**: Please ensure your ChromeDriver version matches your Chrome browser version.
****************************************************************************************************************************
Key Features
- **Smart URL Encoding**: Handles spaces and special characters in keywords automatically.
- **Dynamic Pagination**: Scrapes the total page count from the site to prevent out-of-bounds errors.
- **Visual Automation**: Uses a visible Chrome instance for real-time progress monitoring.
- **Robust Error Handling**: Gracefully manages missing titles or broken links without crashing.
- **Flexible Export**: Choice of Excel (.xlsx) or CSV format for data analysis.
  
Welcome to my 'BBC_NEWS_SCRAPER' README txt.

Tech Stack
- **Language**: Python 3.8
- **Automation**: Selenium WebDriver (Chrome)
- **Data Handling**: Openpyxl, CSV, OS

*The scraper is configured to run in **Visible Mode** to demonstrate the automated browser interaction. Headless mode can be enabled for background processing.*

Improvement & Future Plans
- Implement **Headless Mode** toggle for faster scraping.
- Add **Multi-threading** to speed up multi-page extraction.
- Integrate **Sentiment Analysis** on extracted headlines.
